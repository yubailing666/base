## Ch.14 Chemical Kinetics（化学动力学）

---

### 0. 热力学 vs 动力学

| | **化学热力学** | **化学动力学** |
|:--|:-------------|:-------------|
| 回答 | 反应能否发生？进行到什么程度？热效应如何？ | 反应进行的速率？机理怎么走？如何控制？|
| 本质 | **What can happen?** | **How does it happen?** |

---

### 1. 影响反应速率的四个因素

| 因素 | 说明 |
|:---|:-----|
| **Physical state（物理状态）** | 均相反应通常更快；固体越细（表面积越大）反应越快 |
| **Concentration（浓度）** | 浓度越高 → 碰撞越多 → 速率越快 |
| **Temperature（温度）** | 温度越高 → 分子动能越大 → 有效碰撞越多 |
| **Catalyst（催化剂）** | 改变反应路径（机理），降低活化能，不消耗 |

---

### 2. 反应速率

**定义**：单位时间内浓度的变化量

$$v = -\frac{1}{a}\frac{d[A]}{dt} = \frac{1}{c}\frac{d[C]}{dt} \quad \text{（对反应 } aA \rightarrow cC \text{）}$$

| 类型 | 说明 |
|:---|:-----|
| **Average rate（平均速率）** | 一段时间内的平均变化 |
| **Instantaneous rate（瞬时速率）** | 某一时刻的斜率（反应通常随时间减慢！）|
| **Initial rate（起始速率）** | $t=0$ 时的瞬时速率，化学家最关心的 |

---

### 3. 速率方程（Rate Law）

$$v = k[A]^m[B]^n$$

- $k$ = **rate constant（速率常数）**，温度相关
- $m$, $n$ = **reaction order（反应级数）**，**实验确定**，不等于计量系数！

**重要性质**：
- 速率定律必须由**实验**确定
- 反应级数只针对**反应物**浓度，不针对产物
- 反应级数与化学计量系数**无关**

---

### 4. 积分速率方程与半衰期

#### 零级反应（Zero Order）

$$v = k,\quad [A]_t = [A]_0 - kt$$

$$t_{1/2} = \frac{[A]_0}{2k}$$

线性关系：$[A]$ vs $t$（斜率为 $-k$）

**常见于表面催化反应**（如乙烯加氢、氨在催化剂表面分解）

---

#### 一级反应（First Order）

$$v = k[A],\quad \ln[A]_t = -kt + \ln[A]_0$$

$$t_{1/2} = \frac{0.693}{k}$$

线性关系：$\ln[A]$ vs $t$（斜率为 $-k$）

💡 **一级反应的半衰期与初始浓度无关！**

---

#### 二级反应（Second Order）

$$v = k[A]^2,\quad \frac{1}{[A]_t} = \frac{1}{[A]_0} + kt$$

$$t_{1/2} = \frac{1}{k[A]_0}$$

线性关系：$1/[A]$ vs $t$（斜率为 $k$）

⚠️ **二级反应的半衰期依赖于初始浓度** — 浓度越高，半衰期越短

---

#### 三种反应总结

| 级数 | 微分式 | 积分式 | 线性关系 | 半衰期 |
|:---|:------|:------|:---------|:------|
| **0** | $v = k$ | $[A] = [A]_0 - kt$ | $[A]$ vs $t$ | $[A]_0/2k$ |
| **1** | $v = k[A]$ | $\ln[A] = -kt + \ln[A]_0$ | $\ln[A]$ vs $t$ | $0.693/k$ |
| **2** | $v = k[A]^2$ | $1/[A] = 1/[A]_0 + kt$ | $1/[A]$ vs $t$ | $1/(k[A]_0)$ |

📝 **判断级数的技巧**：看哪个图呈直线！

### 5. 温度与速率 — Arrhenius Equation（阿伦尼乌斯方程）

#### 碰撞理论（Collision Model）

反应发生的三个必要条件：
1. **碰撞**（必要条件，非充分条件）
2. **足够能量**（超过活化能）
3. **有效取向**（有合适的碰撞方向）

#### 活化能（Activation Energy, $E_a$）

**活化能**：反应发生需要克服的最小能量屏障。

**过渡态（Transition State）**：反应过程中能量最高的状态，也称活化络合物（Activated Complex）。

```
反应物 → [过渡态]† → 产物
        ↑ 活化能 Ea
```

反应可以是吸热或放热的，但活化能始终为正。

#### Arrhenius 公式

$$k = Ae^{-E_a/RT}$$

- $A$ = **frequency factor（频率因子）** — 反应物碰撞的频率和取向
- $E_a$ = **activation energy（活化能）**
- $R$ = 气体常数 (8.314 J/mol·K)
- $T$ = 温度 (K)

取对数形式：

$$\ln k = -\frac{E_a}{R}\left(\frac{1}{T}\right) + \ln A$$

**作图求 $E_a$**：$\ln k$ vs $1/T$ → 斜率为 $-E_a/R$

**两点式**（已知两个温度下的 $k$）：

$$\ln\frac{k_2}{k_1} = \frac{E_a}{R}\left(\frac{1}{T_1} - \frac{1}{T_2}\right)$$

📝 **经验规律**：温度每升高 10°C，速率常数大约翻倍。

### 6. 反应机理（Reaction Mechanisms）

**基元反应（Elementary Reaction）**：反应物直接碰撞生成产物的单步反应。

| 分子数（Molecularity） | 说明 |
|:---------------------|:-----|
| **Unimolecular（单分子）** | 一个分子分解或重排，如 $O_3 \rightarrow O_2 + O$ |
| **Bimolecular（双分子）** | 两个分子碰撞，最常见 |
| **Termolecular（三分子）** | 三个同时碰撞，极少见（概率太低）|

**决速步（Rate-Determining Step）**：反应机理中最慢的那一步，决定了总反应的速率。

**一个合理的机理必须满足**：
1. 所有步骤相加 = 总反应方程
2. 决速步能推导出实验测得的速率定律
3. 每一步都平衡
4. 中间体（Intermediate）被产生又被消耗

**例子：快平衡 + 慢步骤**

$$2NO + Br_2 \rightarrow 2NOBr$$
实验测得：$v = k[NO]^2[Br_2]$

机理：
- Step 1（快）：$NO + Br_2 \rightleftharpoons NOBr_2$
- Step 2（慢）：$NOBr_2 + NO \rightarrow 2NOBr$

$v = k_2[NOBr_2][NO]$，由快平衡 $k_1[NO][Br_2] = k_{-1}[NOBr_2]$ 得 $[NOBr_2] = (k_1/k_{-1})[NO][Br_2]$

代入：$v = k_2(k_1/k_{-1})[NO]^2[Br_2] = k[NO]^2[Br_2]$ ✓

### 7. 催化（Catalysis）

**催化剂**：加快反应速率，但不出现在总反应方程式中。**通过改变反应路径降低活化能**。

| 类型 | 说明 |
|:---|:-----|
| **Homogeneous（均相催化）** | 催化剂与反应物在同一相 |
| **Heterogeneous（多相催化）** | 催化剂与反应物在不同相（如固体催化剂 + 气体反应物）|
| **Enzyme（酶催化）** | 生物催化剂，锁钥模型，高度专一 |

**工业应用**：
- 汽车尾气催化转化器
- 合成氨（Haber-Bosch 法）
- SCR 脱硝技术

| 公式 | 用途 |

| $1/[A] = 1/[A]_0 + kt$ | 二级反应积分式 |

| $\ln(k_2/k_1) = E_a/R(1/T_1 - 1/T_2)$ | 双温求 $E_a$ |
