### 1. 介质中的 EM 波

真空中：$c = 3\times 10^8$ m/s。介质中光速变慢：

$$v = \frac{c}{n}$$

**折射率（Index of Refraction）**：
$$n = \frac{c}{v} = \sqrt{K K_m} \approx \sqrt{K}$$

- $K$ = 介电常数（dielectric constant）
- 光通过介质时 **频率不变**，波长变短：$\lambda_{\text{介质}} = \lambda_{\text{真空}}/n$
- $n$ 依赖于频率 → 这就是**色散**的原因

**例题**：钠黄光 $f = 5.09\times 10^{14}$ Hz，钻石 $K=5.84$，求钻石中的光速和波长。
- 真空波长：$\lambda_0 = c/f = 3\times 10^8 / 5.09\times 10^{14} \approx 589$ nm
- 钻石中光速：$v = c/\sqrt{K} = 3\times 10^8 / \sqrt{5.84} \approx 1.24\times 10^8$ m/s
- 钻石中波长：$\lambda = \lambda_0 / n = 589 / \sqrt{5.84} \approx 244$ nm

📝 **考试记住：**
- $f$ 穿过介质不变（光源决定，不是介质决定）
- $\lambda$ 变小（光速变慢了，频率不变，波长自然缩了）

### 2. 偏振（Polarization）

偏振方向定义为 **电场 $\mathbf{E}$ 的振动方向**。

| 类型 | 说明 |
|:---|:-----|
| **Linearly Polarized（线偏振）** | E 在一个固定平面内振动 |
| **Circularly Polarized（圆偏振）** | E 矢量端点做圆周运动（左旋/右旋）|
| **Unpolarized（自然光）** | E 随机方向振动，可分解为两个垂直分量 |

#### 马吕斯定律（Malus's Law）

**经过一个偏振片**：
- 自然光 → 偏振片 → $I_1 = I_0/2$（一半强度丢了）
- 偏振光 → 偏振片（夹角 $\theta$）→ $I = I_0 \cos^2\theta$

**经过两个偏振片**（起偏器 + 分析器）：

自然光 → 偏振片1（$I_0/2$）→ 偏振片2（$I_0/2 \cdot \cos^2\theta$）

#### 经典考题：三个偏振片

```
自然光 → 偏振片A（0°）→ 偏振片B（45°）→ 偏振片C（90°）
```

- A 处：$I_A = I_0/2$
- B 处：$I_B = I_A \cos^2 45^\circ = I_0/2 \cdot 1/2 = I_0/4$
- C 处：$I_C = I_B \cos^2 45^\circ = I_0/4 \cdot 1/2 = I_0/8$

如果**去掉中间**的 B：A（0°）→ C（90°）→ $I = 0$（垂直，全挡）

📝 **反直觉**：中间加一片反而有光透过去！因为中间那片把偏振方向扭转了 45°，让原本垂直的两片有了重叠分量。

#### 考试技巧

- 看见"自然光" → 先减半（$I_0/2$）
- 看见"线偏振光" → 直接套 $I = I_0\cos^2\theta$
- 计算器别忘了：$\cos 45^\circ = \sqrt{2}/2 \approx 0.707$，平方是 0.5

### 3. 反射与折射

#### Snell's Law（斯涅尔定律）

$$n_1 \sin\theta_1 = n_2 \sin\theta_2$$

- $\theta_1$：入射角（相对于法线）
- $\theta_2$：折射角（相对于法线）
- 光从光疏到光密（$n_1 < n_2$）：折射光线**靠近法线**（$\theta_2 < \theta_1$）
- 光从光密到光疏（$n_1 > n_2$）：折射光线**远离法线**（$\theta_2 > \theta_1$）

#### 全内反射（Total Internal Reflection）

条件：光从光密到光疏（$n_1 > n_2$），且入射角超过临界角。

**临界角**：$$\sin\theta_c = \frac{n_2}{n_1}$$

当 $\theta_1 > \theta_c$ → 全部反射回介质中，没有折射光。

💡 应用：光纤通信、内窥镜

#### 考试技巧

- 分清哪边是 $n_1$ 哪边是 $n_2$——错误多从这里来
- 算临界角时 $n_2$ 是光疏介质（折射出去那边的折射率小）
- 全内反射只发生在 $n_1 > n_2$ 的情况下

### 4. 色散（Dispersion）与彩虹

**色散**：折射率 $n$ 依赖于光的波长——不同颜色的光在介质中速度不同。

| 颜色 | 折射率变化 | 偏折程度 |
|:---|:---------|:--------|
| 紫/蓝光 | $n$ 更大 | 偏折更多 |
| 红光 | $n$ 更小 | 偏折最少 |

#### 彩虹的形成

```
太阳光 → 进入水滴（折射）
        → 在水滴内壁一次反射
        → 离开水滴时再次折射（色散分开）
```

- 红光偏折最小 → **外侧**
- 蓝紫光偏折最大 → **内侧**
- 彩虹出现在太阳对面约 $42^\circ$ 视角

**记忆法**：**外红内紫**——和七色光的顺序一致（ROYGBIV，红在波长端外侧）

### 5. 公式速查 & 考试技巧

| 公式 | 用途 |
|:----|:-----|
| $v = c/n$ | 介质中光速 |
| $n = \sqrt{K}$ | 由介电常数算折射率 |
| $n_1 \sin\theta_1 = n_2 \sin\theta_2$ | Snell's Law |
| $\sin\theta_c = n_2/n_1$ | 全内反射临界角 |
| $I = I_0/2$ | 自然光过偏振片 |
| $I = I_0 \cos^2\theta$ | 马吕斯定律（偏振光过偏振片）|

**计算器小抄**：
- $\mu_0 c = 120\pi \approx 377$（真空阻抗，Ch.32 的技法）
- $\sqrt{2}/2 \approx 0.707$（45° 的 sin/cos）
- Snell's law 考试常给 $n$ 值，直接代公式
- 偏振题看清初始光是自然光还是偏振光——决定了要不要先乘 1/2
