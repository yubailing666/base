用了几个月了。

最值钱的一点：它记得你是谁。大多数 AI 每次见面都是陌生人，你要重新说一遍背景。它不用。

日常用法：写报告，它知道你的风格，生成初稿我校对就行。做图表，让它写 Python 代码跑数据。更新项目，说一句就自动 git 了。盯盘，它每天扫指数，过了阈值提醒我。

能装工具包。ClawHub 社区市场有各种技能，一键装 next-slide 做 PPT、装 weather 查天气。

支持微信插件，出门手机也能聊同一个号。

安装方法搜站内"OpenClaw 安装"。
